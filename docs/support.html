常见问题 FAQ

📱 更换手机后，数据会丢失吗？
不会。您的私人日记通过 iCloud 同步，只需在新设备登录相同的 Apple ID，打开应用后数据会自动恢复。请确认设置 → Apple ID → iCloud → 生命金库已开启。
☁️ 如何确认 iCloud 同步是否开启？
前往 iPhone/iPad 系统设置 → 您的名字（Apple ID）→ iCloud → 显示全部 App，找到"生命金库"并确认开关已打开。同步需要网络连接，初次同步可能需要几分钟。
🌐 能量广场加载失败怎么办？
请检查网络连接是否正常。能量广场需要连接互联网才能加载社区内容。若网络正常仍无法加载，应用会显示示例内容，请稍后重试或联系我们。
🗑️ 如何删除已投射到能量广场的内容？
在"财富宝库"中打开对应金库详情页，长按需要删除的日记条目，点击删除图标即可删除本地记录。如需同时撤回社区公开内容，请发邮件至 bhzbtxy@163.com 并提供内容文字，我们将在 3 个工作日内处理。
🔒 我的日记内容安全吗？别人能看到吗？
您的私人日记仅存储在您个人的 iCloud 账户中，采用 Apple 的端对端加密，我们无法访问。只有您选择"投射"到能量广场的内容才会公开，投射前系统会明确提示确认。
👤 如何修改在能量广场显示的昵称？
进入 App 底部"设置"页面，点击头像下方的昵称文字（铅笔图标），即可编辑并保存新昵称。
🖼️ 如何更换头像？
在"设置"页面点击头像圆圈，系统将请求访问相册权限，选择任意图片即可设置为头像。头像仅保存在本地，不上传到服务器。
💎 Pro 会员包含哪些功能？如何升级？
Pro 会员支持添加3个默认金库之外的自定义金库。升级入口在"设置"页面，通过 Apple App Store 内购完成，支持随时取消续订。
🗂️ 如何彻底删除我的所有数据？
1) 删除本地数据：卸载应用即可清除所有本地数据；2) 删除 iCloud 数据：系统设置 → Apple ID → iCloud → 管理账户存储 → 生命金库 → 删除数据；3) 删除社区数据：发邮件至 bhzbtxy@163.com 申请，我们将在 30 个工作日内处理。
🌏 应用支持哪些语言？
当前版本支持简体中文。繁体中文与英文本地化版本正在开发中，敬请期待。
📊 能量广场的"昨日金币"和"金币共振"数据怎么来的？
这两项数据为基于日期算法生成的全球参考数值，每日固定且随日期变化，用于展示社区整体积极能量的趋势，不代表实时精确统计。"我的金库收藏"是您在能量广场收藏的帖子数量。
