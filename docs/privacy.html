隐私政策

PRIVACY POLICY
最后更新：2026年2月  |  Last Updated: February 2026

欢迎使用生命金库（Life Vault）。我们非常重视您的隐私，并致力于以透明、负责任的方式处理您的数据。请仔细阅读本政策，了解我们如何收集、使用和保护您的信息。

本应用遵循"最小必要原则"：仅收集提供核心服务所必需的数据，不收集广告追踪信息，不出售您的任何数据。

一、我们收集的信息

1.1 您主动提供的信息

  • 日记内容：您在各金库中记录的成功日记文字内容（最多30字/条）
  • 昵称：您在设置中自定义的用户名（默认为"生命金库用户"）
  • 头像：您从相册中选择的个人头像图片
  • 社区分享内容：您选择公开投射到能量广场的日记条目

1.2 自动收集的信息

  • 匿名用户标识：App 首次启动时，系统自动生成一个匿名 UUID，用于识别设备和关联您的社区互动记录（点赞、收藏等）。此 ID 不与您的 Apple ID 或任何个人身份挂钩。
  • 互动记录：您在能量广场的点赞、收藏操作，仅用于防止重复操作。

1.3 我们不收集的信息

  • 姓名、电话、电子邮件等个人身份信息
  • 设备位置信息
  • 广告标识符（IDFA）
  • 第三方平台账号信息
  • 浏览行为、崩溃日志（暂未接入第三方统计 SDK）

二、信息的存储与同步

📱

本地存储

头像图片、金库自定义配置、本地收藏列表存储于您设备的本地沙盒中，不上传至任何服务器。

☁️

Apple CloudKit（iCloud 同步）

您的私人日记条目通过 Apple CloudKit 在您本人的 iCloud 账户中加密存储与同步，仅您自己可访问。我们无法读取您的 iCloud 数据。适用 Apple 隐私政策。

🌐

Supabase（能量广场云端服务）

您选择公开投射的日记内容、昵称及匿名用户 ID 存储于 Supabase 托管的数据库中，用于实现能量广场的社区功能。Supabase 服务器位于美国。适用 Supabase 隐私政策。

三、信息的使用目的

  • 提供日记记录、金库管理等核心功能
  • 实现多设备数据同步（通过 iCloud）
  • 展示能量广场社区内容及统计信息
  • 防止重复点赞/收藏等异常操作

我们不会将您的数据用于广告推送、用户画像分析或任何商业化目的。

四、社区内容须知

当您将日记内容"投射"到能量广场时，该内容将以您的昵称公开展示给所有用户。请勿在公开内容中包含个人敏感信息（如姓名、电话、地址等）。一旦投射，该内容将保存于云端服务器，删除请通过应用内"投射记录"进行操作或联系我们。

五、数据保留与删除

  • 私人日记：存储于您的 iCloud，随时可在应用内删除，亦可从 iPhone 设置 → Apple ID → iCloud → 管理账户存储中删除。
  • 社区帖子：可在"能量广场"相关界面删除，或联系我们代为删除。
  • 本地数据：卸载 App 即可清除所有本地数据。
  • 匿名账户：如需删除匿名账户及关联的社区数据，请联系我们，我们将在 30 个工作日内处理。

六、儿童隐私

本应用适合13岁及以上用户使用。我们不会主动收集13岁以下儿童的个人信息。如果您是家长或监护人，发现您的子女在未经授权的情况下使用本应用，请联系我们，我们将立即采取措施。

七、政策更新

我们可能因产品功能更新或法律要求而修订本政策。重大变更将通过应用内通知告知用户。继续使用本应用即表示您同意修订后的政策。
